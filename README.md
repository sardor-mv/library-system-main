# Library-System
- Library Management System 

## Project Description:
- This is a library management system that allows users to add books, delete books, and update books.
- Users can search for books by title, author, and genre and also allows users to borrow books and return books.
- Allows users to view the list of books that are borrowed and the list of books that are available.

## How to run the project:
- Clone the project
- Open the project in your IDE

Make  sure to create an empty database  and change the database name and its credentials in the `application.properties` file to your database name.
Once you have done that, run the project and it will create the tables in your database. And run the "DB Initializer Query File" to insert users and roles into the database.
After doing that you can: 
- Run the project
- Open the browser and type `localhost:8080` in the address bar
- You will be redirected to the login page
- Login with the following credentials:
    - Username: `admin`
    - Password: `changeme`
- You will be redirected to the main page of the project
- You can add books, delete books, update books, search for books, borrow books, return books, view the list of books that are borrowed and the list of books that are available.
